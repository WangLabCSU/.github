We are a specialized [computational biology team](https://faculty.csu.edu.cn/wangshixiang/en/index.htm) at Central South University, driving innovation in bio-intelligence and cancer genomics. Our work focuses on developing computational tools and methods for cancer research, mutational signature analysis, and bioinformatics data processing.

## Tools

### Infrastructure

- [blit](https://github.com/WangLabCSU/blit): Bioinformatics Library for Integrated Tools - Execute command line tool from R
- [geokit](https://github.com/WangLabCSU/geokit): Tools for Accessing and Working with the Gene Expression Omnibus (GEO)
- [xbio](https://github.com/WangLabCSU/xbio): General bioinformatics utilities
- [r-gatk](https://github.com/WangLabCSU/r-gatk): R interface for GATK tools

### Detection

- [mire](https://github.com/WangLabCSU/mire): Microbiome Integrated Reconstruction and Estimation
- [gcap](https://github.com/ShixiangWang/gcap): Gene-level Circular Amplicon Prediction - ecDNA detection from WES data and copy number profiles

### Analysis and/or Visualization

- [sigminer](https://github.com/ShixiangWang/sigminer): 🌲 An easy-to-use and scalable toolkit for genomic alteration signature analysis and visualization in R
- [sigflow](https://github.com/ShixiangWang/sigflow): Streamline Analysis Workflows for Mutational Signatures (Docker/Container)
- [sigminer.prediction](https://github.com/ShixiangWang/sigminer.prediction): Train and Predict Cancer Subtype with Keras Model based on Mutational Signatures
- [bregr](https://github.com/WangLabCSU/bregr): Easy and Efficient Batch Processing of Regression Models in R
- [ezcox](https://github.com/ShixiangWang/ezcox): Easily Process a Batch of Cox Models
- [SigBridgeR](https://github.com/WangLabCSU/SigBridgeR): Screening for phenotype-associated cells at the single-cell level
- [SigBridgeRUtils](https://github.com/WangLabCSU/SigBridgeRUtils): Utilities for SigBridgeR
- [Bizard](https://github.com/openbiox/Bizard): A Biomedical Visualization Atlas <https://openbiox.github.io/Bizard/>
- [metawho](https://github.com/ShixiangWang/metawho): 📦 An R package implementing deft approach for meta analysis
- [neopeptides](https://github.com/ShixiangWang/neopeptides): Calculate and explore property indices of peptides for cancer immunotherapy study

### Data Access

- [faers](https://github.com/WangLabCSU/faers): R interface for FDA Adverse Event Reporting System (Bioconductor)
- [UCSCXenaTools](https://github.com/ropensci/UCSCXenaTools): 📦 An R package for accessing genomics data from UCSC Xena platform (rOpenSci, CRAN)
- [UCSCXenaShiny](https://github.com/openbiox/UCSCXenaShiny): 📊 An R package for interactively exploring UCSC Xena
- [IDConverter](https://github.com/ShixiangWang/IDConverter): Convert Identifiers in Biological Databases
- [rsra](https://github.com/ShixiangWang/rsra): Download NCBI SRA data

### Utilities

- [DoAbsolute](https://github.com/ShixiangWang/DoAbsolute): 📦 Automate Absolute Copy Number Calling using 'ABSOLUTE' package
- [install_GISTIC](https://github.com/ShixiangWang/install_GISTIC): Install GISTIC2 by one line code; GISTIC2 docker image
- [Variants2Neoantigen](https://github.com/ShixiangWang/Variants2Neoantigen): A neoantigen calling pipeline begins from variants record file (MAF)
- [tinyscholar](https://github.com/ShixiangWang/tinyscholar): A Tiny R Package to Get and Show Google Scholar Profile
- [sync-deploy](https://github.com/ShixiangWang/sync-deploy): Shell toolkit for deploying script/command task on remote host
- [contribution](https://github.com/openbiox/contribution): A Tiny Contribution Table Generator Based on ggplot2
