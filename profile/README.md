We are a specialized [computational biology team](https://faculty.csu.edu.cn/wangshixiang/en/index.htm) at Central South University, driving innovation in bio-intelligence and cancer genomics. Our work focuses on developing computational tools and methods for cancer research, mutational signature analysis, and bioinformatics data processing.

## Tools

### Infrastructure

- [blit](https://github.com/WangLabCSU/blit): Bioinformatics Library for Integrated Tools - Execute command line tool from R
- [geokit](https://github.com/WangLabCSU/geokit): Tools for Accessing and Working with the Gene Expression Omnibus (GEO)
- [xbio](https://github.com/WangLabCSU/xbio): General bioinformatics utilities
- [r-gatk](https://github.com/WangLabCSU/r-gatk): R interface for GATK tools

### Detection

- [gcap](https://github.com/ShixiangWang/gcap): Gene-level Circular Amplicon Prediction - ecDNA detection from WES data and copy number profiles ⭐20
- [gcap-wes](https://github.com/ShixiangWang/gcap-wes): GCAP WES pipeline
- [gcaputils](https://github.com/ShixiangWang/gcaputils): Utils for GCAP package

### Analysis and/or Visualization

- [sigminer](https://github.com/ShixiangWang/sigminer): 🌲 An easy-to-use and scalable toolkit for genomic alteration signature analysis and visualization in R ⭐157
- [sigflow](https://github.com/ShixiangWang/sigflow): Streamline Analysis Workflows for Mutational Signatures (Docker/Container)
- [sigminer.prediction](https://github.com/ShixiangWang/sigminer.prediction): Train and Predict Cancer Subtype with Keras Model based on Mutational Signatures
- [faers](https://github.com/WangLabCSU/faers): R interface for FDA Adverse Event Reporting System (Bioconductor) ⭐38
- [bregr](https://github.com/WangLabCSU/bregr): Easy and Efficient Batch Processing of Regression Models in R ⭐14
- [SigBridgeR](https://github.com/WangLabCSU/SigBridgeR): Screening for phenotype-associated cells at the single-cell level
- [SigBridgeRUtils](https://github.com/WangLabCSU/SigBridgeRUtils): Utilities for SigBridgeR
- [ezcox](https://github.com/ShixiangWang/ezcox): Easily Process a Batch of Cox Models ⭐23
- [regport](https://github.com/ShixiangWang/regport): Regression Model Processing Port
- [metawho](https://github.com/ShixiangWang/metawho): An R package implementing deft approach for meta analysis
- [polar](https://github.com/ShixiangWang/polar): Dots and Their Connections in Polar Coordinate System (ggplot2 extension)

### Data Access

- [UCSCXenaTools](https://github.com/ropensci/UCSCXenaTools): 📦 An R package for accessing genomics data from UCSC Xena platform (rOpenSci, CRAN) ⭐111
- [copynumber](https://github.com/ShixiangWang/copynumber): The "copynumber" R package with support for hg38 ⭐20
- [IDConverter](https://github.com/ShixiangWang/IDConverter): Convert Identifiers in Biological Databases

### Utilities

- [DoAbsolute](https://github.com/ShixiangWang/DoAbsolute): 📦 Automate Absolute Copy Number Calling using 'ABSOLUTE' package ⭐41
- [install_GISTIC](https://github.com/ShixiangWang/install_GISTIC): Install GISTIC2 by one line code; GISTIC2 docker image ⭐41
- [Variants2Neoantigen](https://github.com/ShixiangWang/Variants2Neoantigen): A neoantigen calling pipeline begins from variants record file (MAF) ⭐30
- [tinyscholar](https://github.com/ShixiangWang/tinyscholar): A Tiny R Package to Get and Show Google Scholar Profile
- [sync-deploy](https://github.com/ShixiangWang/sync-deploy): Shell toolkit for deploying script/command task on remote host ⭐21
- [sync2gitee](https://github.com/ShixiangWang/sync2gitee): Auto-Sync GitHub Repositories to Gitee ⭐25
